新增功能

最新版本
auto-flip="ms"
翻到背面後自動倒數翻回正面，底部顯示進度條。值為毫秒數。
toggle
布林屬性。翻到背面後點擊任意處即可翻回正面，背面顯示「↺ 點擊翻回」提示。
flip-bar-color="name"
單獨指定倒數進度條的顏色。若未設定，自動跟隨 color-after / color。

使用方式

<!-- ① auto-flip：3 秒後自動翻回 -->
<card-toggle
  content="這是背面內容"
  auto-flip="3000"
  color="info">
  點我，3 秒後自動翻回
</card-toggle>

<!-- ② toggle：點擊背面可翻回 -->
<card-toggle
  content="這是背面內容"
  toggle
  color="lavender">
  點我切換，背面再點可翻回
</card-toggle>

<!-- ③ 同時使用：5 秒自動翻回，背面也可提前點擊翻回 -->
<card-toggle
  content="背面內容"
  auto-flip="5000"
  toggle
  color="safe">
  5 秒自動翻回，或點背面提前翻回
</card-toggle>

<!-- ④ flip-bar-color：單獨指定進度條顏色 -->
<card-toggle
  content="背面"
  auto-flip="4000"
  color="lavender"
  flip-bar-color="warning">
  進度條顏色與卡片顏色不同
</card-toggle>

flip-bar-color 接受所有內建顏色名稱，也接受任意 CSS 色彩值（如 #ff6600、rgb(255,100,0)）。

auto-flip 與 toggle 可單獨或同時使用。兩者並用時，倒數期間點擊背面可提前翻回，進度條同步停止。
快照機制已修正為 Lazy Snapshot：正面內容在第一次點擊前才記錄，確保所有瀏覽器（含 Firefox、Safari）子節點已完整解析，翻回正面後內容不會空白。

What is this?

card-toggle 是一個 Web Component（<card-toggle>），可將靜態卡片轉為互動元件。支援三種模式：

  • Reveal — 點擊後替換為任意 HTML 內容
  • Quiz — 點擊後顯示答案輸入框，驗證後給予回饋
  • Group — 以 <card-toggle-group> 包覆多張卡片，支援堆疊或輪播導覽

搭配的 CardToggle.php 可直接從 PHP 變數或資料庫查詢結果產生卡片標記，自動處理 ID 生成與 HTML 跳脫。

需要 Bootstrap Icons 提供測驗模式的按鈕圖示。針對桌機寬螢幕版面最佳化。

Files

your-project/
├── card-toggle.js      ← Web Component（CSS 自動注入）
├── CardToggle.php      ← PHP 渲染類別
└── your-page.php       ← 你的頁面，require CardToggle.php

Quick Start

<!-- Bootstrap Icons（測驗模式按鈕需要） -->
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/font/bootstrap-icons.min.css">

<!-- Reveal：點擊顯示內容 -->
<card-toggle color="info" source="detail-1">
  點擊查看詳情
</card-toggle>

<div id="detail-1" style="display:none">
  <p>這是揭露後的內容，可以是任意 HTML。</p>
</div>

<!-- 在 </body> 前引入腳本 -->
<script src="card-toggle.js"></script>
<?php
require_once 'CardToggle.php';

// 從資料庫列揭露
echo CardToggle::revealArray('王大明　點擊展開', [
    '部門' => $emp['dept'],
    '職稱' => $emp['title'],
    '過敏' => $emp['allergy'],
], ['color' => 'lavender']);

// 測驗模式
echo CardToggle::quiz(
    '阿斯匹靈最大單次劑量是多少 mg？',
    $drug['max_single'],
    '<p>正確！' . $drug['dosage'] . '</p>',
    ['color' => 'warning', 'max_attempts' => 3]
);

echo CardToggle::script('/assets/card-toggle.js');
?>

HTML Usage

Reveal mode

三種提供內容的方式，每張卡片擇一使用。

content 屬性 — 內嵌 HTML 字串（適合短內容）：

<card-toggle color="safe" content="這是說明文字，可含 <b>HTML</b>">
  點擊查看
</card-toggle>

source 屬性 — 指向隱藏元素（適合大型 HTML 區塊）：

<card-toggle color="info" source="my-detail" animation="slide">
  點擊查看詳情
</card-toggle>

<div id="my-detail" style="display:none">
  <table>...</table>
</div>

無屬性 — 卡片本體即為觸發器，適合群組內使用。

Quiz mode

加入 question 屬性啟用測驗模式，點擊後顯示輸入框，驗證答案並給予回饋。

<card-toggle
  color="warning"
  question="阿斯匹靈最大單次劑量是多少 mg？"
  answer="650"
  max-attempts="3"
  hint="介於 325 到 650 之間"
  hint-target="hint-area"
  success-source="aspirin-answer"
  success-color="safe"
  auto-next>
  阿斯匹靈最大單次劑量是多少 mg？
</card-toggle>
<div id="aspirin-answer" style="display:none">正確！325–650 mg，每 4–6 小時一次。</div>
<div id="hint-area"></div>

Group

以 <card-toggle-group> 包覆多張卡片，支援堆疊或輪播。

<!-- Stack 模式（預設）-->
<card-toggle-group mode="stack" theme="sky">
  <card-toggle color="sky"  source="step-1">Step 1</card-toggle>
  <card-toggle color="info" source="step-2">Step 2</card-toggle>
</card-toggle-group>

<!-- Slide 模式 -->
<card-toggle-group mode="slide" theme="warning" show-pages>
  <card-toggle color="warning" question="題目一" answer="A">題目一</card-toggle>
  <card-toggle color="warning" question="題目二" answer="B">題目二</card-toggle>
</card-toggle-group>

PHP Class Usage

CardToggle::reveal()

揭露模式，HTML 內容自行組裝。escape 預設 false。

echo CardToggle::reveal(
    '點擊查看診斷結果',
    '<table>...' . nl2br(htmlspecialchars($row['note'], ENT_QUOTES, 'UTF-8')) . '</table>',
    ['color' => 'info', 'animation' => 'slide']
);

CardToggle::revealText()

純文字資料庫欄位揭露。escape 強制 true，htmlspecialchars + nl2br 自動套用。

echo CardToggle::revealText(
    'Step 1 確認病患身分　點擊展開',
    $sop['description'],
    ['color' => 'sky', 'number' => 1, 'animation' => 'slide']
);

CardToggle::revealArray()

直接傳入資料庫列，key 成為標籤，value 自動跳脫，輸出定義清單版面。

echo CardToggle::revealArray(
    '王大明　點擊展開',
    [
        '部門'     => $emp['dept'],
        '職稱'     => $emp['title'],
        '藥物過敏' => $emp['allergy'],
    ],
    ['color' => 'lavender', 'animation' => 'fade']
);

CardToggle::quiz()

echo CardToggle::quiz(
    $drug['name'] . ' 的最大單次劑量是多少 mg？',
    $drug['max_single'],
    '<p>正確！' . htmlspecialchars($drug['dosage'], ENT_QUOTES, 'UTF-8') . '</p>',
    [
        'color'         => 'warning',
        'max_attempts'  => 3,
        'placeholder'   => '輸入數字（mg）',
        'hint'          => $drug['dosage'],
        'hint_target'   => 'quiz-hint-area',
        'error_message' => '答案不正確，請再想想',
        'success_color' => 'safe',
        'auto_next'     => true,
    ]
);

CardToggle::groupOpen() / groupClose()

echo CardToggle::groupOpen([
    'mode'       => 'slide',
    'theme'      => 'warning',
    'show_pages' => true,
]);
foreach ($questions as $q) {
    echo CardToggle::quiz($q['question'], $q['answer'], '', [
        'color'    => 'warning',
        'auto_next' => true,
    ]);
}
echo CardToggle::groupClose();

// 在 </body> 前輸出 script tag（多次呼叫只輸出一次）
echo CardToggle::script('/assets/card-toggle.js');

Escape & Newline Handling

Methodescape 預設\n → <br>適用時機
reveal()false手動自行組裝 HTML
revealText()true（強制）✅ 自動純文字資料庫欄位
revealArray()true（強制）✅ 自動直接傾倒資料庫列
quiz()false手動問題與成功內容來自資料庫
card()false手動純展示靜態卡片
使用 reveal()、quiz()、card() 自行組裝 HTML 時，務必對不可信欄位進行跳脫：
htmlspecialchars($value, ENT_QUOTES, 'UTF-8')
含換行：nl2br(htmlspecialchars($value, ENT_QUOTES, 'UTF-8'))
